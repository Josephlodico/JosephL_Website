## ▶️ How to Run

### Option 1 — Open Directly (quickest)
Just double-click `index.html` in your file explorer. It will open in your default browser with no setup needed.

### Option 2 — Live Server (recommended for development)
This project is already configured for VS Code's Live Server on port **5501**.

1. Open the project folder in **VS Code**
2. Install the [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) if you haven't already
3. Right-click `index.html` → **"Open with Live Server"**
4. Your browser will open at `http://127.0.0.1:5501`

> Live Server auto-refreshes the page whenever you save a file, which is great for making edits.

### Option 3 — Any Local HTTP Server
If you prefer another tool (e.g. Python):
```bash
# Python 3
python -m http.server 5501
```
Then visit `http://localhost:5501` in your browser.
