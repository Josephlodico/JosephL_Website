MyGlidingSim (Info)

MyGlidingSim is A glider simulation game that i tried making for my Game Engine 2 class. What made me think about doing a gliding simulator game for a project was this popular game called fortnite which is what insprired me to try and make this gliding simulator myself.

MyGlidingSim / Level Design (Level 1)

The level design is based on floating platforms in the air so that the player can decide where to go and reach the goal to go to the next level (real level picture on hold for now)
