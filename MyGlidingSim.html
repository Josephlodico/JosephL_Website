MyGlidingSim

Description

MyGlidingSim is A glider simulation game that i tried making for my Game Engine 2 class. What made me think about doing a gliding simulator game for a project was this popular game called fortnite which is what insprired me to try and make this gliding simulator myself.

Project Info

  • Role: Gampelay, Level Design,UI Programmer
  • Team size: 1
  • Engine: Unreal Engine 5
  • Tools: Perforce
  • Duration: 1 Month
  • Date: March and April 2025
  • Theme: Simulator

MyGlidingSim LOGO

Here is The Logo For MyGlidingSim, I made this logo Using Canvas because i wanted to make my logo more Unique instead of using the Unreal engine Splash. And the design shows an Umbrella in the back of the title since its one of the main Mechanics in the Game.

MyGlidingSim / Level Design (Level 1)

The level design is based on floating platforms in the air so that the player can decide where to go and reach the goal to go to the next level.

MyGlidingSim / Level Design (Level 2)

In Level 2 there is not that much of a difference but there are more Floating platforms that you can land on. However, I'm Gonna try and improve Level 2 since a did not have that much time to think about the level design.

MyGlidingSim Checkpoints

In Level 1 and Level 2 there are Checkpoints for the player to mark as there safe spot. However, i think there is only a checkpoint in Level 1 and not Level 2 and i think they also don't work properly. so i think i will fix it in the near future.
